# BeachBot AI – Smart Beach Cleaning Robot

## Project Overview

BeachBot AI is an autonomous robot designed to improve beach cleaning efficiency.  
The goal is to reduce manpower, operational cost, and environmental pollution through automation.

This project was developed for a robotics competition and focuses on applying AI and system design to a real-world problem.

## Problem

Beach environments face several challenges:

- Large amount of waste accumulation  
- High manpower and operational costs  
- Poor user experience due to pollution  

---

## Solution
<img src="./images/solution.png" width="500" />

BeachBot AI provides an automated cleaning system with:

- Garbage detection using AI  
- Autonomous navigation using LiDAR  
- Waste collection and classification  
- Support for renewable energy

## Technology

The project applies AI and robotics technologies, including:

- YOLO (You Only Look Once) for real-time object detection  
- LiDAR for mapping and obstacle avoidance  
- Computer vision for waste identification  
- Embedded system for robot control  

### YOLO Object Detection
<img src="./images/yolo.png" width="200" />

YOLO is used to detect different types of garbage in real time.  
The model processes camera input and identifies objects, enabling the robot to locate waste efficiently during operation.


### LiDAR Navigation
<img src="./images/lidar.png" width="200" />


LiDAR is used to scan the environment and generate a map of the area.  
Based on the map, the robot follows a predefined path and avoids obstacles in real time.
